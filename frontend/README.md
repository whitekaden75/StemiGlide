# Stemi Glide Web Application

## Overview
This project is a React + TypeScript web application designed to showcase and support the Stemi Glide product. The application demonstrates a product-focused website with interactive elements, structured data, and a workflow demo tailored to real EMT use cases.

The goal of this project is to combine frontend development (React, TypeScript) with practical product design, including user interaction, media integration, and basic backend/data concepts.

---

## Features
- Product overview and marketing pages
- Step-by-step EMT workflow demonstration
- Image gallery and media integration
- Video demonstration section
- Navigation between multiple pages using React Router
- Structured data-driven components (cards, timelines, etc.)
